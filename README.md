# ISC Wario Charger — Releases

Public download + auto-update channel for the **ISC Wario Charger** desktop app
(AMS pit display & charge controller for the IFS08).

The app's source is maintained in a private repository; this repo only hosts the
released installers so the app's built-in update check can reach them.

## Download

Grab the latest installer from the [**Releases**](../../releases/latest) page:

| Platform | File |
|----------|------|
| 🍎 macOS (universal) | `*_universal.dmg` |
| 🪟 Windows | `*_x64-setup.exe` |
| 🐧 Linux (Debian/Ubuntu) | `*_amd64.deb` |
| 🐧 Linux (Fedora/RHEL) | `*.x86_64.rpm` |
| 🐧 Linux (any distro) | `*_x86_64.AppImage` |

The app checks this repo on startup and offers an in-app update when a newer
version is published here.
